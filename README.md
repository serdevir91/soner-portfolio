## Features

*   **Print Friendly**: Optimized CSS for printing / saving as PDF.
*   **Tech Stack**: Built with React + Vite.

## Setup
1.  Install dependencies:
    ```bash
    npm install
    ```
2.  Run development server:
    ```bash
    npm run dev
    ```

## Deployment
To deploy to GitHub Pages:
```bash
npm run deploy
```
